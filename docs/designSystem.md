# Design System

This document elaborates on the colors, fonts, and font sizes used universally in  the **Ovuli** app.

### Colors
The primary colors used are:
```sh
#F78161
#F54C64
```
The secondary colors used are:
```sh
#878E88
#0C1120
#E6EFFB
#F54C64
```
